# tools

## Script

- [trej.ts](https://github.com/miy4/tools/blob/main/trej.ts) Auto-detects Japanese or English input and translates it to the other language using a Google Gemini API
- [dropper.ts](https://github.com/miy4/tools/blob/main/dropper.ts) Converts a web page into a Markdown document
- [chatgpt_chatlog_to_md.py](https://github.com/miy4/tools/blob/main/chatgpt_chatlog_to_md.py) Converts a ChatGPT export JSON file into Markdown files
- [http-server.bash](https://github.com/miy4/tools/blob/main/http-server.bash) Starts a simple local HTTP server in the current directory on a specified port
- [touchp.bash](https://github.com/miy4/tools/blob/main/touchp.bash) `mkdir -p` + `touch`
